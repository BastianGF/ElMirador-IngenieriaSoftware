# ElMirador-IngenieriaSoftware: Informe 3
Proyecto semestral - Sistema de Administración Edificio El Mirador - Ingeniería de Software

## Contenido
- Prototipo interactivo desarrollado en Figma / Figma Make
- Evaluación heurística de Nielsen (ISO 25010)
- Registro de casos de prueba
- Control de versiones
- Presentación

## Herramientas utilizadas
- Figma Make (generación IA del prototipo)
- Figma Design (refinamiento y navegación)
- GitHub (repositorio y control de versiones)

Se adoptó versionamiento semántico (MAJOR.MINOR.PATCH) combinado con el historial nativo de Figma. GitHub actúa como repositorio central de todos los artefactos del proyecto.
